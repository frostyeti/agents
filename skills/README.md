# Skills

This directory contains skills for agents. Skills are reusable capabilities that agents can use to perform tasks.

## Structure

Each skill should be organized in its own subdirectory with appropriate documentation and implementation files.

## Adding a New Skill

1. Create a new directory for your skill
2. Add a README.md describing the skill's purpose and usage
3. Include any implementation files needed for the skill

## Examples

Skills might include:
- Code analysis and review
- Documentation generation
- Test creation
- Debugging assistance
- And more...
