# agents

A repository for agent files and skills that can be cloned into your home directory.

## Installation

Clone this repository into your home directory:

```bash
git clone https://github.com/frostyeti/agents.git ~/.agents
```

## Project Structure

```
~/.agents/
├── skills/          # Agent skills directory
├── README.md        # This file
└── LICENSE          # MIT License
```

## Skills

The `skills/` directory contains reusable capabilities for agents. See [skills/README.md](skills/README.md) for more information on adding and using skills.
